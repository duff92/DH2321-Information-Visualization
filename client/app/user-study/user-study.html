User study

Steam graph

1) Would like to see more of the graph, because when opening it on the Macbook Pro laptop you cannot see the whole brushing tool. Feels like it's only a reflection of the graph rather than a brushing tool (see Image 1).

Solution: Changed the footer to not have fixed position, as the brushing tool then was hided on smaller screens.

2) Difficult to understand what the different colors represent in the graph. Especially, because the info text showing up in the left corner is quite small and just shows quickly.

Solution: Added more explaining data in the info box at the left top

3) Cannot understand the meaning of the number zero in the graph. It feels like it should be 4500 according to the user, as the graph is generated in the middle of the viewport, which makes it confusing (see Image 2).

4) One user tried to interact with the main graph initially (but nothing happened), and said it was difficult to see how to use the controller below the graph.

5) Finally, it was difficult to see when you switched between different days (by moving the mouse horizontally). Therefore, one suggestion was to show the interaction steps that was possible to make.

Solution: Added a thin vertical line that followed the mouse when it hovered the graph.

Parallel coordinates

Impossible to read the text on smaller screen sizes (see Image 3). And would have like to be able to clear the brushed areas.

Also (this for a user without any previous experience of parallel coordinates), the instructions of how to interact with the visual structure was difficult to grasp. The lines in the graph was hard to relate to any data, which was as a response to the question of what the lines represented. The first impression when interacting with the graph (brushing) was that a lot of data showed in the table below.
